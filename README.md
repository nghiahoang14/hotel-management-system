# Hotel Management System

A full-stack hotel management web application built with **Next.js** and **NestJS**.

## Project Structure

hotel-management-system
│
├── hotel-management       (Frontend - Next.js)
└── hotel-management-be    (Backend - NestJS)

## Features
- Browse hotel rooms
- View room details
- Make reservations
- Booking history
- Admin dashboard for managing rooms, bookings, and users
## Run Project

### Backend
npm run start:dev  

### Frontend
npm run dev  
## Technologies

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
- Axios

### Backend
- NestJS
- MongoDB
- Mongoose
- JWT Authentication
- Cloudinary
- Nodemailer
